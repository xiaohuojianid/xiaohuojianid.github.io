VPN 配置添加完成后即可使用 --- ## 🔗 如何添加节点
✍️

手动添加

  1. 点击右上角 +
  2. 选择节点类型
  3. 填写地址、端口、密码
  4. 点击「完成」保存
📋

订阅导入 推荐

  1. 点击右上角 +
  2. 类型选择 Subscribe
  3. 粘贴订阅链接
  4. 保存后自动拉取节点
📷

扫码添加

  1. 点击左上角扫描图标
  2. 扫描节点二维码
  3. 自动识别并添加
--- ## 🔌 首次连接 1. 选择一个节点 2. 打开顶部的连接开关 3. 首次连接会提示 VPN 授权，点击 **「允许」** 4. 连接成功后状态栏会显示 VPN 图标 --- ## 🎯 进阶配置 ### 规则配置 Shadowrocket 支持强大的规则分流功能： | 模式 | 说明 | |------|------| | **全局路由** | 所有流量走代理 | | **配置（规则）** | 按预设规则分流 | | **场景模式** | 不同网络环境自动切换 | ### 如何导入 Clash 配置 1. 将 Clash 配置文件的链接转换为 Shadowrocket 兼容格式 2. 点击右上角 **+** → Subscribe 3. 粘贴订阅链接 4. 保存并更新 ### Sing-box 节点导入 Shadowrocket 2.0 以上版本支持 Sing-box 订阅格式，直接导入即可使用。 --- ## ❓ 常见配置问题 ### 连接后无法上网？ 1. 检查节点是否在线 2. 尝试切换路由模式为「全局路由」 3. 确认 VPN 配置未被其他应用冲突 {% include tip-box.html type="warning" title="订阅更新失败？" content="检查订阅链接是否过期，尝试使用蜂窝数据更新，或重新导入订阅链接。" %}
